# Lookup Example for Solvers

## Introduction
This example shows how to implement a lookup for linear solvers used within the WRIMS engine.
More information about the Netbeans Lookup API can be found here: 
[NetBeans Lookup](https://netbeans.apache.org/wiki/main/netbeansdevelopperfaq/DevFaqLookup/).

## Cache
The example solver lookup implementation uses a cache to reduce the number of lookups.

## Global Singletons
Using the `Lookup.getDefault()` method provides access to the global instance singletons. 
See more here: 
[Default Lookup](https://netbeans.apache.org/wiki/main/netbeansdevelopperfaq/DevFaqLookupDefault/).
